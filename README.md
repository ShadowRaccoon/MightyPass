# Mighty pass

Pequeño programa para evaluar qué tan segura es una contraseña


## Dependencias:
- tqdm

## Ejemplo modo de uso: 

### Ejecución normal: (Igreso de contraseña escondido)
```
python main.py
```

### Ejecución parametrizada: (Ingres)
```
python main.py -p password -o resultado.txt
```
